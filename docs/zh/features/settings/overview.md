## 功能概述

设置模块是 ModxAI Studio 的核心配置中心，提供应用层面的全局配置能力。主要特点：

- **一键环境安装**：降低用户配置门槛，自动检测硬件并推荐最佳环境
- **差异化依赖加载**：按需安装，未使用的功能不安装依赖，节省磁盘空间
- **沙箱式动态加载**：环境隔离，仅在需要时加载对应环境，避免冲突
- **多语言支持**：支持中文和英文界面切换
- **主题切换**：多套预设主题，支持浅色/深色模式
- **授权管理**：支持终身版和订阅版授权激活

## 环境配置

环境配置是 ModxAI Studio 的**核心亮点功能**之一，旨在降低用户的环境配置门槛。

### 设计理念

- **零门槛安装**：用户无需手动配置 Python、PyTorch 等复杂环境
- **差异化加载**：按需安装依赖，训练功能需要时才安装训练相关包
- **沙箱隔离**：环境完全独立，不依赖系统 Python，避免冲突
- **断点续传**：安装中断后可继续，已下载的包会保留

### 系统信息

环境配置区域会自动检测并显示当前系统信息：

| 信息项 | 说明 |
|--------|------|
| 系统 | 操作系统类型（Windows）|
| CPU | CPU 核心数 |
| 内存 | 系统总内存（GB）|
| GPU | 显卡类型（NVIDIA/AMD）|
| 驱动 | NVIDIA 驱动版本（仅 NVIDIA GPU 显示）|

### 基本系统要求

系统会检测以下环境是否满足要求：

| 检测项 | 说明 |
|--------|------|
| 平台 | 操作系统兼容性 |
| Python | 内置便携式 Python 环境 |
| 内存 | 建议 16GB 以上 |
| 磁盘 | 需要足够的安装空间 |
| GPU | 可选，有 NVIDIA GPU 可启用 GPU 加速 |
| NVIDIA 驱动 | GPU 环境需要 |
| CUDA 版本 | GPU 环境需要 |

### 安装控制

| 操作 | 说明 |
|------|------|
| **环境类型选择** | 选择 CPU 或 GPU 环境（GPU 需有 NVIDIA 显卡）|
| **镜像源选择** | 官方（pypi.org）/ 阿里云 / 腾讯云，根据所在地区的网络环境选择镜像源 |
| **刷新** | 重新检测环境状态 |
| **清理** | 删除已安装的环境依赖 |
| **安装 / 重装** | 开始安装或重新安装环境 |

### 安装流程

1. 选择环境类型（CPU 或 GPU）
2. 选择镜像源（中国内地用户建议选择阿里云或腾讯云）
3. 点击"安装"按钮
4. 等待安装完成（显示进度条和日志）
5. 安装成功后，相关功能模块即可使用,无需重启应用

### 功能可用性

安装完成后，右上角会显示功能可用性图标：

| 功能 | 说明 |
|------|------|
| 模型管理 | 模型加载、卸载等基础功能 |
| 数据处理 | 数据集创建和处理功能 |
| 训练 | 模型微调和训练功能 |

> 详细的环境安装指南请参阅 [环境安装详解](./environment.md)

## 常规设置

常规设置用于配置应用的基本行为：

### 选择语言 / Select language

| 选项 | 说明 |
|------|------|
| English | 英文界面 |
| 中文 (简体) | 简体中文界面 |

切换语言后，整个应用界面会立即更新为对应语言。

### 关闭行为 / Close behavior

设置点击窗口关闭按钮时的行为：

| 选项 | 说明 |
|------|------|
| **最小化到托盘** | 关闭窗口时最小化到系统托盘，应用继续在后台运行 |
| **直接退出** | 关闭窗口时直接退出应用 |
| **询问操作** | 每次关闭时弹出对话框询问（默认）|

### 开机启动 / Auto start

| 状态 | 说明 |
|------|------|
| 开启 | 系统启动时自动启动 ModxAI Studio |
| 关闭 | 需要时手动启动（默认）|

### 最小化托盘 / Minimize to tray

| 状态 | 说明 |
|------|------|
| 开启 | 最小化时隐藏到系统托盘|
| 关闭 | 最小化时保留在任务栏 （默认）|

## 全局参数

全局参数影响应用的运行时行为，对所有功能模块生效。

### 推理设备 / Inference device

选择模型推理时使用的计算设备：

| 选项 | 说明 |
|------|------|
| **自动** | 自动检测并选择最佳设备（默认）|
| **仅 CPU** | 强制使用 CPU 进行推理 |
| **仅 GPU** | 强制使用 GPU 进行推理（需已安装 GPU 环境）|

> 💡 **提示**：如果安装了 GPU 环境，推荐使用"自动"选项，系统会根据GPU实际情况选择最佳设备。

### 最大令牌数 / Max tokens

设置全局的最大输出令牌数限制：

| 属性 | 值 |
|------|-----|
| 最小值 | 1 |
| 最大值 | 8192 |
| 默认值 | 4096 |
| 步进 | 32 |

> 用于限制数据处理时每条数据输出的最大 token 数。若超过训练时的序列长度，会截断超出部分。

### 脏词敏感度 / Badwords sensitivity

设置内容过滤的严格程度：

| 选项 | 说明 |
|------|------|
| **关闭** | 不进行任何过滤（默认）|
| **低敏感度** | 基础过滤，仅过滤明显违规内容 |
| **严格模式** | 增强安全过滤 |
| **阻断模式** | 检测到敏感词立即终止处理 |

## 主题设置

ModxAI Studio 提供多套预设主题，支持浅色和深色模式，用户可根据喜好自由切换。

### 切换方式

1. 在主题设置区域浏览主题预览卡片
2. 点击想要使用的主题卡片
3. 主题立即生效，选择会自动保存

## 许可证

许可证区域用于查看授权状态和激活注册码。

### 授权类型

| 类型 | 说明 |
|------|------|
| **免费版** | 高级功能会消耗免费额度/月 |
| **订阅版** | 按月/年订阅，无限制使用全部功能 |
| **终身版** | 一次购买，永久使用 |

### 激活方式

1. 在输入框中输入注册码
2. 点击"激活"按钮
3. 激活成功后显示授权状态

### 状态说明

| 状态 | 说明 |
|------|------|
| 激活成功 | 授权有效，可无限制使用全部功能 |
| 已过期 | 订阅已到期，需续订 |
| 授权失效 | 注册码已被吊销，请联系支持 |

> 详细的授权说明请参阅 [授权激活详解](./license.md)
