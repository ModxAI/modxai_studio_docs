# 记忆管理

> v1.1.0
> 2026-05

记忆管理用于统一查看和管理智能体在运行过程中沉淀的**长期记忆**，以及用户手动**收藏**的问答。您可在此预览、编辑、归档、删除记忆，或将记忆导出为可供训练或知识库使用的数据格式。

## 记忆来源与分类

记忆管理中的记录来自两个渠道：

| 类别 | 来源 | 说明 |
|------|------|------|
| **智能体记忆** | Agent 运行中自动生成 | 智能体在执行任务时自主保存的高价值数据, 适合导出成知识库数据|
| **手动收藏** | 用户在对话中手动标记 | 用户对智能体或普通聊天的回答进行 👍 反馈保存的偏好/长记忆，以及手动收藏的问答对, 适合导出成训练数据 |

### 记忆类型

不同记忆类型在智能体内部有不同的用途：

| 类型 | 说明 | 管理页可见 |
|------|------|-----------|
| `insight` | 从对话中提炼的高价值数据 | ✅ |
| `favorite_qa` | 收藏问答（训练数据） | ✅ |

## 浏览与筛选

### 统计面板

页面顶部展示当前记忆的统计数据：

| 统计项 | 说明 |
|--------|------|
| 总记忆数 | 当前筛选条件下的记忆总数 |
| 智能体记忆 | 智能体自动沉淀的记忆数量 |
| 手动收藏 | 用户手动收藏的记忆数量 |
| 可导出 SFT | 支持导出为 SFT 格式的记忆数量 |
| 可导出 RAG | 支持导出为 RAG 格式的记忆数量 |
| 已归档 | 全量数据中已归档的记忆数量 |

### 筛选条件

您可以通过以下筛选条件快速定位记忆：

| 筛选项 | 可选值 | 说明 |
|--------|--------|------|
| **搜索框** | 自由文本 | 按标题、摘要、正文或问答内容搜索 |
| **类别** | 全部 / 智能体记忆 / 手动收藏 | 按记忆来源筛选 |
| **记忆类型** | 动态列表 | 按具体记忆类型筛选（如 insight、favorite_qa 等） |
| **归档状态** | 未归档 / 全部 / 已归档 | 筛选活跃或已归档的记忆 |

## 查看记忆详情

在列表中点击某条记忆的**预览**按钮，弹出详情面板，展示该记忆的完整信息：

| 字段 | 说明 |
|------|------|
| 标题 | 记忆的标题 |
| 类别 | 智能体记忆 / 手动收藏 |
| 类型 | 记忆的具体类型 |
| 标签 | 记忆关联的标签列表 |
| 导出状态 | 已导出 / 未导出 |
| 归档状态 | 活跃 / 已归档 |
| 问题（收藏记忆） | 收藏问答中的问题 |
| 回答（收藏记忆） | 收藏问答中的回答 |
| 摘要（智能体记忆） | 记忆摘要 |
| 正文内容（智能体记忆） | 记忆完整内容 |
| 创建时间 | 记忆生成的时间 |
| 更新时间 | 最近修改的时间 |
| 最近导出时间 | 最近一次导出的时间 |

## 编辑记忆

> **注意**：编辑后**直接覆盖**原记忆内容，不可恢复，请确认后再保存。

点击列表中的**编辑**按钮，弹出编辑弹窗。

根据记忆类别，编辑表单会显示不同字段：

| 类别 | 可编辑字段 |
|------|-----------|
| 智能体记忆 | 标题、摘要、正文内容 |
| 手动收藏 | 标题、问题、回答 |

保存后会要求二次确认，确认后更新记忆内容。

## 归档与取消归档

### 归档

归档后的记忆对智能体**不可见**，但仍在管理页保留，可查看和导出。

- **单个归档**：点击记忆的归档按钮，确认后归档
- **批量归档**：勾选多条记忆后，点击**归档所选**

### 取消归档

取消归档后，记忆重新对智能体可见。

- **单个取消归档**：在已归档视图中点击取消归档按钮
- **批量取消归档**：勾选多条已归档记忆后，点击**取消归档**

> **注意**：归档操作仅对来源为「智能体记忆」的记录生效。收藏问答不参与归档。

## 删除记忆

> **删除后不可恢复**，请谨慎操作。

点击记忆的**删除**按钮，确认后删除。

## 导出记忆

您可以将记忆导出为数据文件，用于后续的**模型微调（SFT）**或**知识库构建（RAG）**。

### 前置条件

- 至少选择一条记忆记录
- 指定导出名称和输出目录
- 所选记录需与导出格式兼容（见下方说明）

### 导出步骤

1. 在列表中勾选要导出的记忆
2. 点击**导出**按钮，弹出导出设置面板
3. 填写导出名称
4. 选择输出目录
5. 选择导出格式：
   - **SFT**：导出为问答训练数据，适用于模型微调
   - **RAG**：导出为可直接用于**生成向量**步骤的数据，适用于知识库构建

### 导出兼容性

| 导出格式 | 兼容条件 | 不兼容情况 |
|----------|---------|-----------|
| **SFT** | 仅支持手动收藏的记忆 | 智能体自动记忆不兼容 |
| **RAG** | 具有正文内容的记忆 | 仅有标题无正文的记忆不兼容 |

> **提示**：导出面板会实时显示「可导出」和「将跳过」的数量，帮助您判断所选记录是否适合当前格式。

导出成功后，可选择是否**打开输出目录**查看导出文件。

### 导出状态

记忆被导出后会标记为「已导出」，后续再次导出时不会重复标记为未导出。

---

## 注意事项

- 搜索支持模糊匹配标题、摘要、正文和问答内容
- 归档操作对智能体的记忆检索行为有直接影响 —— 已归档意味着智能体在上下文中不会召回该记忆
- SFT 导出仅支持「手动收藏」类别，智能体自动记忆如果内容适合微调，需先通过收藏功能转化为收藏记忆
- RAG 导出产出的是可直接用于**生成向量**步骤的数据
