# 预设管理

> v1.1.0
> 2026-05

预设管理允许您保存当前工作台的完整参数配置，并在需要时快速恢复。适用于在不同任务类型之间切换配置的场景。

---

## 什么是预设

预设是工作台参数的**完整快照**，包含以下内容：

- 所有角色的参数配置（模型、提示词、推理参数等）
- 高级参数设置
- 工作区目录和 Skills 目录
- 指令设置（全局指令文件路径、个人偏好）

---

## 保存预设

1. 在右侧配置面板中调整好所有参数
2. 点击**预设管理**按钮
3. 在预设对话框中点击**保存当前配置**
4. 输入预设名称
5. 确认保存

系统内置了默认预设（不可删除），新安装后首次使用时自动创建。

---

## 加载预设

1. 点击**预设管理**按钮
2. 在预设列表中找到目标预设
3. 点击该预设，选择**加载**
4. 确认后，当前工作台参数将替换为预设中的配置

加载预设后，当前会话的配置会被覆盖。建议在加载前确认当前配置已保存。
