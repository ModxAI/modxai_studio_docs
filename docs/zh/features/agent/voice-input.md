# 语音输入

> v1.1.0
> 2026-05

智能体支持通过麦克风进行语音输入，系统自动将语音转写为文本并填入输入框。

## 前置条件

> ⚠️ **语音输入必须先加载 STT 模型和接入麦克风设备**，否则功能不可用。

### 所需模型

| 模型 | 用途 | 要求 |
|------|------|------|
| STT 模型 | 将语音转为文字 | 必须加载才能使用语音输入 |
| TTS 模型 | 将文字转为语音（朗读回复） | 可选，仅朗读功能需要 |

### 模型加载

1. 在**模型库**中导入或下载**音转文**模型
2. 加载**音转文**模型
3. 如需朗读功能，同时加载**文转音**模型

> 未加载 音转文 模型时，点击麦克风会提示「语音识别模型未就绪」。

## 使用方法

### 开始录音

1. 点击输入框左侧的**麦克风按钮**
2. 系统开始录音，按钮状态变为录音中
3. 界面显示「正在录音」提示

### 结束录音并转写

1. 再次点击麦克风按钮结束录音
2. 系统自动将录音发送给 STT 模型进行转写
3. 界面显示「正在转写」提示
4. 转写完成后，文本自动填入输入框

### 录音时长

- 单次录音最长 **120 秒**（2 分钟）
- 超时后自动结束录音并开始转写

### 取消录音

在录音过程中，可通过取消操作放弃当前录音（不进行转写）。

## 状态说明

语音输入有以下几种状态：

| 状态 | 界面提示 | 说明 |
|------|---------|------|
| 空闲 | 「点击麦克风开始语音输入」 | 等待用户开始录音 |
| 录音中 | 「正在监听语音输入」 | 正在录制音频 |
| 转写中 | 「正在转写语音输入」 | 正在将音频转为文字 |
| 转写完成 | 文本自动填入输入框 | 可编辑后发送或直接发送 |
| 空结果 | 「未识别到有效语音内容」 | 录音中未检测到有效语音 |
| 转写失败 | 「语音转写失败」 | STT 模型处理出错 |

## 朗读回复

智能体的回复支持**朗读**功能（TTS 语音合成）。

### 前置条件

- 必须加载 **TTS（语音合成）模型**
- 未加载时朗读按钮不可见

### 使用方法

点击智能体回复旁的**朗读按钮**（🔊），系统将回复内容合成为语音并播放。

## 注意事项

### 模型就绪检查

- STT 模型未就绪时，麦克风按钮不可用
- TTS 模型未就绪时，朗读按钮不可用
- 模型加载状态在模型库页面查看

### 录音质量

- 确保麦克风设备正常工作
- 在安静环境中录音可获得更准确的转写结果
- 支持系统默认麦克风设备

### 转写准确性

- 转写准确性取决于 STT 模型质量和录音清晰度 推荐使用faster-whisper-large-v3模型
- 转写结果会自动填入输入框，发送前可手动编辑修正
- 不满足转写准确性时可尝试更换 STT 模型
