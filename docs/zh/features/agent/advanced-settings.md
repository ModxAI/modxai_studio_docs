# 高级参数

> v1.1.0
> 2026-05

高级参数用于精细调整智能体的行为策略。所有参数均有合理默认值，一般用户无需修改。高级用户可根据需要调整。

## 参数分组

高级参数按功能分为以下 10 个分组，在界面中以折叠面板形式展示：

| 分组 | 说明 |
|------|------|
| 安全 | 安全审查开关与命令黑名单 |
| 工具全局 | 影响所有工具的全局限制 |
| 文件读取 | 文件读写相关限制 |
| 网页搜索 | 搜索引擎相关参数 |
| 浏览器 | 浏览器操作相关参数 |
| 爬虫 | 网站爬取相关参数 |
| 内容缓存 | 上下文内容缓存策略 |
| 在线模型 | 在线 API 调用策略 |
| 上下文压缩 | 对话压缩阈值控制 |
| 评估节点 | 回答评估重试策略 |

## 工具全局

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 工具执行超时 | 600 秒 | 0.1 ~ 3600 秒 | 单次工具调用的最大执行时长。超时后工具调用将被中断。 |
| 页面正文上限 | 5000 | 1000 ~ 100000 字符 | 网页、浏览器、爬虫提取正文时的最大字符数。 |
| 默认分页行数 | 100 | 50 ~ 5000 行 | 读取文件时的默认分页行数。 |
| 重复输出阈值 | 20 | 1 ~ 200 | 连续出现相同工具输出多少次后触发保护机制。 |
| 重复调用阈值 | 50 | 1 ~ 100 | 连续重复调用同一工具的最大次数，超过后将暂停执行。 |

## 上下文压缩

上下文压缩参数控制对话历史在什么条件下触发压缩，以及压缩后各优先级内容的比例分配。

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| P0 比例 | 0.2 | 0 ~ 1 | 核心系统提示词与当前任务的预算占比 |
| P1 比例 | 0.2 | 0 ~ 1 | 工具结果与任务状态的预算占比 |
| P2 比例 | 0.4 | 0 ~ 1 | 对话历史的预算占比 |
| P3 比例 | 0.2 | 0 ~ 1 | 摘要、知识索引与记忆的预算占比 |
| 压缩触发比例 | 0.8 | 0.1 ~ 0.9 | 压缩时模型输入占上下文窗口的比例 |

## 安全

### 命令黑名单

- **默认值**：预设常用危险命令列表
- **说明**：禁止执行的终端命令列表。支持精确匹配、命令前缀匹配与通配符 `*` `?`。基础命令会自动覆盖其参数变体（如 `rm` 可匹配 `rm file.txt`），同时可识别链式命令、管道、cmd/powershell/bash 包装命令中的真实子命令。
- **格式**：用逗号或换行分隔多个命令

默认黑名单包含 `rm`、`del`、`format`、`shutdown`、`taskkill` 等常用危险命令。可根据需要增减。

## 参数修改须知

- 所有高级参数修改需点击**应用**按钮生效
- 超出范围的数值将被限制在允许的 min/max 区间内
- 可通过**预设管理**保存和恢复整套参数配置
