# 附件与知识库

> v1.1.0
> 2026-05

智能体支持在发送消息时附加文件、图片、音频等资源，以及连接 RAG 知识库。本文介绍各类附件的使用方法与限制。

## 附件类型

输入框左侧的附件菜单支持以下类型：

| 类型 | 图标/入口 | 说明 |
|------|----------|------|
| 图片 | 附件菜单 → 图片 | 上传图片供智能体分析 |
| 音频 | 附件菜单 → 音频 | 上传音频文件供处理 |
| 文件 | 附件菜单 → 文件 | 上传任意文件供读取 |
| 视频 | 附件菜单 → 视频 | 上传视频文件 |
| 本地附件 | 附件菜单 → 本地附件 | 综合入口，支持多种格式 |
| 知识库连接 | 附件菜单 → 知识库连接 | 关联 RAG 知识库 |

## 本地附件

### 添加附件

1. 点击输入框左侧的附件按钮
2. 选择对应的附件类型
3. 从本地文件系统中选择文件

### 支持格式

| 附件类别 | 支持格式 |
|---------|---------|
| 图片 | JPG、PNG 等常见图片格式 |
| 音频 | 常见音频格式 |
| 文件 | 文本、PDF、Word、代码文件等 |
| 视频 | 常见视频格式 |

### 使用说明

- 附件会随消息一起发送给智能体
- 智能体可读取附件内容进行分析
- 附件按对话隔离，切换会话时附件列表会自动恢复
- 多模态模型支持图文理解（需加载多模态模型类别）

## 知识库连接

将 RAG 知识库关联到当前会话，智能体可优先基于您的私有知识库内容回答问题。

### 前置条件

- 已在创建或导入 RAG 知识库
- 建议同时加载向量和重排序模型,提高检索质量

### 连接知识库

1. 在附件菜单中选择**知识库连接**
2. 在弹出的对话框中，从下拉列表选择已创建的知识库
3. 确认后，已连接的知识库名称会显示在输入框旁

### 断开知识库

点击已连接知识库名称旁的关闭按钮即可断开。

### 知识库参数

连接知识库时可配置以下参数：

| 参数 | 说明 |
|------|------|
| 检索方法 | 混合（推荐）/ 向量 / 向量+重排 / 文本 |
| 检索数量 | 检索返回的文档数量 |
| 重排序候选数 | 可能提升召回质量 |
| 分数阈值 | 相似度过滤（0~1） |
| 最大上下文长度 | 单次检索的字符数上限 |
| 检索提示词 | 自定义检索行为指令 |
