# 智能体工作台

> v1.1.0
> 2026-05

智能体工作台是 ModxAI Studio 的核心功能模块。与普通对话不同，智能体会**自主拆解任务、选择工具、逐步执行并生成最终结果**——您只需描述目标，智能体会自动完成检索网络、读取文件、执行命令、生成报告等一系列操作。

## 环境要求

智能体工作台对模型和环境有**最低配置要求**。不满足以下条件将无法正常使用。

### 模型要求

> **必须为「主角色」配置模型**，否则无法创建新会话或发送消息。

| 项目 | 最低要求 | 推荐配置 |
|------|---------|---------|
| 模型系列 | Qwen3 / Qwen3.5 等较新模型 | Qwen3-8B 或以上 |
| 参数规模 | ≥ 4B | 8B~32B |
| 上下文长度 | ≥ 32K tokens | 64K~128K |
| 量化格式 | Q4_K_M | Q_8_0 |

> **为什么有这些要求？** 智能体需要同时理解任务目标、拆解步骤、阅读工具输出并做出决策，小模型或短上下文的模型容易陷入循环或无法完成多步任务。

模型来源可以是：
- **本地模型**：在模型库中下载并加载（支持 LLM /VLM / HF 标准模型）
- **在线模型**：通过 API Key 连接云端服务商（支持 DeepSeek、OpenRouter 等）
- **局域网模型**：连接局域网内的模型服务

### 浏览器与 Web 搜索环境

智能体的**浏览器操作**和**网页搜索**功能依赖额外的 Python 库。如需使用这些工具，请在「设置 → 环境配置」中安装智能体环境依赖。

- 若未安装相关依赖，浏览器和搜索工具不能使用.
- 仅使用文件读写、终端命令等基础功能无需额外安装

> Web搜索执行时,建议中国内地用户使用网络代理.否则可能导致搜索失败

### 语音输入环境

语音输入需要加载 **STT（语音识别）** 和 **TTS（语音合成）** 模型。详见 [语音输入](./voice-input.md)。

## 快速开始

### 第一步：配置主角色模型

1. 打开智能体工作台页面
2. 在右侧**会话配置**面板中，展开「角色参数」→「主角色」
3. 选择连接方式（本地 / 在线 / 局域网）并指定模型
4. 模型选择后即可开始使用

> 未配置主角色模型时，**新建会话**按钮和**发送**按钮将被禁用，界面会提示「主角色未配置模型，无法创建新会话」。

### 第二步：创建会话

点击左侧**新建会话**。系统会自动生成临时标题（如「新会话-10:30」），首次发送任务后自动更新为正式标题。

### 第三步：描述任务

在输入框中用自然语言描述您想让智能体完成的目标。例如：
- 「帮我检索关于 AI 最新进展的网络信息，并整理成一份摘要报告」
- 「读取我工作区中的 README.md，提炼核心内容并润色成一封邮件」
- 「分析 data.csv 文件，生成一份统计报告」

按 `Enter` 发送，按 `Shift + Enter` 换行。

## 核心概念

### 角色体系

智能体使用**多角色协作**架构，每个角色承担不同的职责：

| 角色 | 说明 | 是否必须 |
|------|------|---------|
| **主角色** | 负责主要推理、工具编排与最终回复 | ✅ 必须 |
| **评估角色** | 审查正式回复质量，提供反馈与修改建议 | 默认启用，可选 |
| **摘要角色** | 上下文压缩与辅助总结 | 可选 |
| **记忆角色** | 基于微调模型进行模糊记忆搜索 | 可选 |
| **安全角色** | 高风险内容前置审查与安全建议 | 可选 |
| **委派角色** | 将子任务委派给独立配置的角色执行 | 可选 |

各角色的模型和参数可独立配置。除主角色外，其他角色若未指定模型，会自动复用主角色的模型。详见 [角色配置](./roles.md)。

### 工具集

智能体可调用多种工具来完成您的任务：

| 工具类别 | 包含能力 |
|---------|---------|
| **文件操作** | 读取、写入、移动、搜索文件 |
| **终端执行** | 运行命令和脚本 |
| **网页搜索** | 搜索引擎检索、网页内容读取 |
| **浏览器** | 启动浏览器、导航页面、截图、交互操作 |
| **爬虫** | 批量抓取网站内容 |
| **待办计划** | 生成、更新任务待办列表 |
| **委派协作** | 将子任务分派给其他角色执行 |
| **记忆管理** | 搜索、读取、保存长期记忆以及运行期提醒 |
| **交互** | 询问用户及建议 |
| **计算机** | 屏幕截取 |
| **多媒体** | 图像生成、视频生成、语音合成 |

可以在右侧面板的「工具集」中为每个角色选择允许使用的工具。


### 高级参数

高级参数允许您精细调整智能体的行为，例如工具执行超时、网页搜索限制、浏览器超时、安全策略等。详见 [高级参数](./advanced-settings.md)。

## 会话管理

### 新建会话

点击左侧**新建会话**。系统自动生成临时标题，首次发送消息后根据任务内容自动更新为正式标题。

### 切换会话

点击左侧会话列表中的任意会话即可切换。切换时会自动恢复该会话的历史消息和执行进度。

### 重命名会话

点击会话右侧菜单，选择**重命名**，输入新标题后保存。

### 删除会话

点击会话右侧菜单，选择**删除**。删除后不可恢复。

### 归档会话

对于暂时不需要查看但仍想保留的会话，可以使用**归档**功能。归档后的会话从主列表中隐藏，可通过**归档管理**查看和还原。支持单个和批量操作。

## 发送消息

### 基本发送

在输入框中输入任务描述，按 `Enter` 发送。智能体会自动开始分析并执行。

### 发送模式

- **发送到新会话**：在当前会话中新建一轮对话，将消息作为新任务发送
- **补充到当前会话**：将消息追加到当前任务中，智能体会基于已有上下文继续执行

## 消息操作

### 编辑消息

对于已发送的用户消息，可进行编辑：

1. 点击消息右侧的**编辑**按钮
2. 修改消息内容
3. 保存后更新

### 删除消息

支持两种删除范围：

- **仅本条**：删除当前消息
- **本条及后续**：删除当前消息及其之后的所有消息

> 智能体正在运行时，不能删除已被处理的消息。

### 重发消息

对消息执行**重发**操作时，会删除该消息及其后续内容，然后重新发送。

### 复制消息

点击**复制**按钮将消息内容复制到剪贴板。

### 反馈记忆

对智能体的回复可使用以下反馈：

- **偏好记忆**（👍）：将回复标记为满意的记忆，供后续参考
- **规避记忆**（👎）：将回复标记为需要规避的模式

反馈会自动保存为长期记忆，影响后续的智能体行为。

---

### 附件

发送消息时可附加以下内容，智能体会自动读取和分析：

| 附件类型 | 说明 |
|---------|------|
| **本地附件** | 图片、音频、视频、文件 |
| **知识库连接** | 关联已创建的 RAG 知识库 |
| **工作区引用** | 引用其它工作区的目录(只读) |

详见 [附件与知识库](./attachments.md)。

### 提示词优化

帮助您快速生成高质量的提示词，提升与AI对话的效果。详见 [提示词优化](../chat/prompt-generation.md)

### 语音输入

支持通过麦克风进行语音输入，系统自动将语音转写为文本。**需要加载 STT 模型**。详见 [语音输入](./voice-input.md)。

## 执行过程

### 实时进度

智能体执行任务时，界面会实时显示：

- **进度节点**：每个分析步骤、工具调用的状态和结果
- **待办面板**：当前任务的待办列表，实时更新完成进度
- **思考过程**：开启「思考模式」后可见 AI 的推理逻辑
- **状态提示**：底部状态栏显示「正在处理」「正在压缩对话」「正在评估」等实时状态

### 自动审批

开启**自动审批**后，智能体遇到需要确认的操作（如执行终端命令）时将自动放行，无需手动确认。

> ⚠️ 自动审批适用于您信任智能体的场景。关闭时，高风险操作会暂停等待您确认。详见 [自动审批与安全](./auto-approval.md)。

### 暂停与恢复

以下情况智能体会暂停等待您的决策：
- 执行可能修改文件或系统的命令
- 安全审查检测到风险内容
- 工具调用轮次达到上限
- AI 请求人工选择或决策
- 模型暂时没有有效进展

暂停时可选择**继续**或**停止**。

### 上下文压缩

当对话长度超过模型上下文窗口时，智能体会自动压缩历史消息，提取关键信息后继续执行，确保长任务不会因上下文不足而中断。

### 回滚操作

对于智能体在工作区中创建或修改的文件，支持**回滚**到修改前的状态。点击修改节点旁的**回滚**按钮，确认后恢复。

## 注意事项

### 模型配置

- **必须先配置主角色模型**才能创建会话和发送消息
- 主角色支持本地/在线/局域网三种连接方式，任选其一即可
- 在线模型需先在**凭据管理**中添加 API Key
- 记忆角色强制使用本地模型，不支持在线或局域网方式
- 摘要/安全/评估/委派角色若未配置模型，会自动使用主角色的模型

### 工具依赖

- **浏览器**和**网页搜索**需要安装智能体环境依赖（在设置中安装）未安装时仅相关工具不能使用，不影响其他功能
- **TTS 语音合成**和**STT 语音识别**需要加载对应模型
- **多媒体工具**需要加载对应模型,否则无法得到有效结果

### 执行安全

- 智能体可执行终端命令和修改文件，请在可信任的工作区中使用
- 关闭**自动审批**可以防止意外的高风险操作
- 可在高级参数中配置**命令黑名单**，禁止特定命令的执行
- 启用**安全角色**可增加额外的安全检查层

### 性能建议

- 本地模型建议使用同期较新模型(如Qwen3.5 4B+), 上下文长度设置建议32K+, 能更好的完成执行智能体任务.

### 工作区

- 未指定工作区目录时，系统使用临时目录下的默认工作区
- 智能体的文件操作默认在工作区范围内进行
- 可在会话配置中自定义工作区目录
