## ModxAI Studio 是什么？

ModxAI Studio 是一款专为**本地化AI部署**打造的一站式桌面端应用。将本地模型部署、自主智能体、图像生成、知识库检索、数据处理和模型训练等功能整合到应用中，让普通用户以**零成本** 、**高效灵活** 、**不受限制**的使用智能体完成日常任务,不必担心数据泄露.

**核心定位**：与市面上依赖云端大模型的智能体应用不同，ModxAI Studio 的智能体框架专为**本地小模型**深度优化。通过精细的状态管理和容错机制，即使 1B-3B 参数的小模型也能在 32K 上下文内完成日常任务.如果与在线模型配合使用也能大幅节省token的开销

## 核心功能

### 🧠 自主智能体

让 AI 不只是聊天，而是能**自主执行任务**的生活助手：

- **任务规划与执行**：AI 自动分解复杂任务，逐步执行并反馈进度
- **内置工具链与外部技能挂载**：支持内置70+高效工具和外部Skill接入,满足网页浏览、本地文件读写编辑、执行终端命令等多种端到端任务场景.
- **记忆与偏好学习**：交互式管理记忆和独有的自动化记忆训练，打造真正越用越懂你的模型分身.

### 🤖 AI 对话

与本地大语言模型进行智能对话：

- **文本对话**：日常问答、代码生成、创意写作等
- **多模态对话**：支持图文理解，上传图片让 AI 识别分析
- **RAG 增强**：基于您的知识库回答问题，AI 更懂您的领域
- **数据收藏**：收藏优质对话用于后续训练

### 🖼️ 图像与视频生成

使用 Stable Diffusion 模型进行创意生成：

- **文生图**：根据文字描述生成精美图像
- **图生图**：基于现有图片进行风格转换或内容修改
- **视频生成**：支持 Wan 系列等视频生成模型

### 🎤 TTS 语音合成

使用本地 TTS 模型进行文本转语音：

- **中英双语**：支持中文和英文文本的语音合成
- **多音色选择**：提供多种预设音色供选择
- **语速调节**：可调整语音播放速度
- **自动语言检测**：根据输入文本自动识别语言

### 📚 模型库管理

统一管理各类 AI 模型：

- **多模型支持**：聊天模型、多模态模型、HF、向量模型、重排模型、SD 模型、TTS 模型、STT 模型
- **一键下载**：从 Hugging Face 或 ModelScope 直接下载热门模型
- **本地导入**：支持 GGUF、SafeTensors 等多种格式
- **LoRA 适配**：为模型加载专属微调适配器
- **局域网服务**：加载后自动暴露 OpenAI 兼容 API，供局域网内其他应用调用
- **在线模型服务**: 目前开放DeepSeek和OpenRouter在线模型服务.

### 📊 数据处理

将原始数据转化为可训练数据集：

- **SFT 数据生成**：从文档、结构化数据等生成监督微调数据
- **RAG 知识库**：构建向量化知识库，支持智能检索
- **音频转文本**：将语音转录为多种结构化文本
- **AI 合成**：通过本地模型按需完成数据分析

### 🎯 模型训练

- **LoRA 高效微调**：仅需 8GB 显存即可开始训练
- **实时监控**：可视化训练进度和损失曲线
- **检查点评估**：自动评估多个检查点，推荐最优结果
- **一键导出**：将训练成果导出为 GGUF 格式，直接用于推理
- **记忆训练**：基于智能体主动积累或与用户手动收藏的对话记忆，自动迭代优化专属模型

### ⚙️ 零门槛环境配置

告别复杂的环境配置：

- **一键安装**：自动检测硬件，推荐最佳配置
- **按需选择**：支持全部/智能体/语音合成三种安装能力，按需安装
- **沙箱隔离**：独立环境，不影响系统其他软件
- **按需加载**：功能未使用时不加载依赖

## 系统要求

### 最低配置

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11 (64位) |
| 内存 | 16GB RAM |
| 存储 | 20GB 可用空间 |
| 处理器 | 支持 AVX2 指令集 |

### 推荐配置

| 项目 | 要求 |
|------|------|
| 内存 | 32GB+ RAM |
| 显卡 | NVIDIA RTX 系列 (8GB+ 显存) |
| 存储 | SSD 固态硬盘 |

> 💡 无 GPU 也可使用 CPU 模式运行，适合入门体验。
